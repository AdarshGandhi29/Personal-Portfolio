# Personal Portfolio

Welcome to my **Personal Portfolio**! This project showcases my professional journey, skills, projects, and achievements in a sleek and responsive web design.

## 🖥️ Demo

Experience the portfolio live: [Personal Portfolio](https://adarshgandhi29.github.io/Personal-Portfolio/)

## 📜 Features

- **About Me**: A brief introduction and background.
- **Projects**: Highlights of my key projects with descriptions and links.
- **Skills**: Showcase of my technical proficiencies.
- **Contact**: Easy-to-use contact form and social media links.
- **Responsive Design**: Accessible on all devices, including mobile and desktop.
